# Security Policy

## Reporting a Vulnerability

Security issues should be reported directly to the maintainer at armontidubosehill@gmail.com. This inbox is actively monitored.

**Do not open a public issue for security vulnerabilities.**

## Response Timeline

- Acknowledgment within 48 hours
- Initial assessment within 5 business days
- Resolution timeline communicated after assessment

## Scope

This policy applies to the Iter MCP server and associated SDKs in this repository.

For detailed security model and threat documentation, see [docs/SECURITY.md](docs/SECURITY.md).
